# 🔋 PULSE - System Cleaner & Monitor

A lightweight, fast, and ad-free system utility for Windows that combines cleaning, monitoring, and a built-in handbook.

![Pulse Icon](https://github.com/epopovo816-boop/PULSE-CLEANER-or-monitoring-/assets/263048529/a59bef9e-697c-4172-9aed-dde1beb47ad5)

---

## ✨ Features

### 🧹 **System Cleaner**
- Remove unnecessary files and temporary data
- Free up disk space efficiently
- One-click cleanup

### 📊 **System Monitoring**
- Real-time **CPU** usage tracking
- **GPU** performance monitoring
- **Temperature** sensors
- **Internet** connection monitoring
- System resource dashboard

### 📚 **Built-in Handbook**
- Quick reference guide
- System information
- Useful tips and tricks

---

## 📋 Requirements

- **OS:** Windows 7 and above
- **RAM:** 512 MB (recommended: 2 GB+)
- **Disk Space:** 40 MB
- **No admin rights required** (some features may need elevation)

---

## 🚀 Quick Start

1. **Download** the latest release: [Pulse.exe v1.0.0](https://github.com/epopovo816-boop/PULSE-CLEANER-or-monitoring-/releases/tag/1.0.0v)
2. **Run** the executable file
3. **Enjoy** fast, ad-free system monitoring and cleaning!
