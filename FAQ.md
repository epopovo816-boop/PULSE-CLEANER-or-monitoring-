# ❓ Frequently Asked Questions (FAQ)

## General Questions

### What is PULSE?
PULSE is a lightweight, ad-free system utility for Windows that helps you monitor system performance, clean temporary files, and manage your computer's health. It's completely free and weighs only 40 MB.

### Is PULSE really free?
Yes! PULSE is 100% free, forever. No ads, no paid features, no tracking. We believe good software should be accessible to everyone.

### Does PULSE contain malware or viruses?
No. PULSE is open-source and safe. You can view the source code on GitHub to verify this yourself. We recommend scanning any downloaded file with your antivirus software for extra peace of mind.

### What are system requirements?
- **OS:** Windows 7 and above (Windows 10/11 recommended)
- **RAM:** 512 MB minimum (2 GB+ recommended)
- **Disk Space:** 40 MB
- **Internet:** Optional (not required for core features)

### Does PULSE require administrator rights?
Not for basic features. Some advanced features (like cleaning system files) may require elevation. PULSE will ask for permission when needed.

## Installation & Usage

### How do I install PULSE?
1. Download the latest release from [Releases](https://github.com/epopovo816-boop/PULSE-CLEANER-or-monitoring-/releases)
2. Run the `Pulse.exe` file
3. No installation needed - it's portable!

### Is PULSE portable?
Yes! PULSE is a standalone executable. You can run it from a USB drive or any folder without installation.

### Can I uninstall PULSE?
Simply delete the `Pulse.exe` file. Since it's portable, there's no installation to remove.

### How often should I use the cleaner?
It depends on your usage:
- **Light users:** Once a month
- **Regular users:** Once a week
- **Heavy users:** 2-3 times per week

Start with a weekly cleaning and adjust based on how much junk accumulates on your system.

### Is it safe to delete files with the cleaner?
PULSE is designed to safely remove temporary files, cache, and junk data that Windows no longer needs. However:
- Always backup important data first
- Review what will be deleted before proceeding
- Start with non-critical files

## Features & Performance

### What can the System Cleaner do?
The cleaner removes:
- Temporary files
- Cache data
- Old logs
- Duplicate files
- Unused registry entries
- Browser cache and cookies

### Does cleaning improve performance?
Yes, but the impact depends on how much junk is on your system. You may notice:
- Faster startup times
- More available disk space
- Smoother operation (in some cases)

### Can PULSE monitor my system in real-time?
Yes! The monitoring feature shows live statistics for:
- CPU usage
- GPU performance
- Temperature sensors
- Internet connection status
- RAM usage
- Disk space

### Is the temperature monitoring accurate?
Temperature accuracy depends on your hardware sensors. Most modern systems report accurate temperatures, but some older hardware may have calibration issues.

### What is the DNS tool for?
The DNS tool allows you to:
- Change your DNS provider
- Diagnose DNS issues
- Improve internet speed (in some cases)
- Resolve connectivity problems

## 🔊 Voice Alerts

### Can I disable voice alerts?
Yes! You can turn off alerts in the settings menu. You can re-enable them anytime.

### How often do alerts play?
Critical alerts play once per minute if the issue persists. This prevents notification spam while keeping you informed. For example, if your CPU is overloaded, you'll hear a warning every minute until the issue is resolved.

### Is there a test alert feature?
Yes! You can test the alert system to verify your audio is working correctly before relying on it during system issues. This is useful when you first set up PULSE.

### What kind of alerts does PULSE have?
Currently, voice alerts warn about:
- **CPU Overload** - When processor usage is critically high
- **Temperature Issues** - When your system is getting too hot

### In what languages are alerts available?
Currently, voice alerts are available in:
- Russian (Русский)
- English

### Why not more languages yet?
Adding voice alerts requires:
- Professional voice recordings
- Quality audio processing
- Testing across different systems

We're planning to add more languages in future releases! If you'd like to help by providing voice recordings, please open an Issue to discuss.

## Troubleshooting

### PULSE won't start
- Try running as administrator
- Check if antivirus is blocking it
- Download the latest version
- Restart your computer

### The cleaner says "Access Denied"
- Run PULSE as administrator
- Close all other programs
- Try again

### Temperatures show as 0°C
Your hardware may not report temperature data. This is normal for some systems and doesn't indicate a problem.

### Internet monitoring shows incorrect speeds
- Check your actual connection speed at speedtest.net
- Restart your router
- The tool shows real-time usage, not max speed

### PULSE uses too much RAM
PULSE is designed to be lightweight. If you notice high RAM usage:
- Update to the latest version
- Close unnecessary programs
- Restart PULSE

### I can't hear voice alerts
- Check your system volume
- Make sure speakers/headphones are connected
- Use the test alert feature to verify audio
- Check if alerts are disabled in settings
- Make sure your audio device is set as default

## Security & Privacy

### Does PULSE collect my data?
No. PULSE doesn't track you, collect data, or phone home. It's completely offline and operates locally on your computer.

### Is my data safe when using the cleaner?
Yes. PULSE only removes files that:
- Windows marks as temporary
- Are no longer needed by the system
- You can recover from the Recycle Bin (if enabled)

### Can I trust PULSE?
Yes! PULSE is open-source. You can view the entire source code on GitHub to verify what it does.

### What about my passwords and files?
PULSE never touches:
- Personal documents
- Photos and videos
- Program files or settings
- Saved passwords or credentials
